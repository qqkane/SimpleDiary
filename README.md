# Simple Diary
- Что это такое? - Это простенький макет электронного дневника.
- Что делает ? - Позволяет выставлять уроки в дневнике на любые дни недели от пн до сб.
- Для чего ? - Для удобного управления уроками в течении недели.
- Как работает? - Вы нажимаете кнопку **Admin Panel**, далее выбираете в нужной ячейке урок который вы хотите поставить в данный день недели из выпадающего списка и нажимаете **сохранить**.
