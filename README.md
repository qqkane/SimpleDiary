# Simple Diary
- Что это такое? - Это простенький макет электронного дневника.
- Что делает ? - Позволяет выставлять уроки в дневнике на любые дни недели от пн до сб.
- Для чего ? - Для удобного управления уроками в течении недели.
- Как работает? - Вы нажимаете кнопку **Admin Panel**, далее выбираете в нужной ячейке урок который вы хотите поставить в данный день недели из выпадающего списка и нажимаете **сохранить**. Далее после сохранения вы можете посмотреть список уроков от лица ученика.
- Как установить? - Переходите на страницу https://github.com/qqkane/SimpleDiary/tree/main и нажимаете Code, далее Download ZIP 
- Как запустить? Установите Python 3.12 с официального сайта и разархивируйте скачанный .zip архив, далее запустите файл main.py и пользуйтесь программой!
